OSI Model
	ISO-International Oraganization of Standardization
	OSI-Open Systems Interconnection model
	ISO-OSI model-
		Developed in 1984
	Basic elements of Layers
		Services
			A set of actions that a layer offers to higher layer
		Protocols
			A set of rules that a layer uses to exchange information
		Interfaces
			Communication b/w layers
	
	7 Layers
		Applications
			Host layers
			Data Unit-data
			Netowrk process to application
			Devices/apps-webserver, mailserver, browser
			Protocols-http, https,ftp, ssh, telnet, dns, pop3, dhcp, ntpsnmp
		Presentation
			Host layers
			Data Unit-data
			Data representation & Encryption
			Devices/apps-webserver, mailserver, browser
			Protocols-http, https,ftp, ssh, telnet, dns, pop3, dhcp, ntpsnmp
		Session
			Host layers
			Data Unit-data
			Interhost communication
			Devices/apps-webserver, mailserver, browser
			Protocols-http, https,ftp, ssh, telnet, dns, pop3, dhcp, ntpsnmp
		Transport
			Host layers
			Data Unit-segments
			End-to-End connections & Reliability
			Devices/apps-gateway
			Protocols-tcp, udp
		Network
			Media Layers
			Data Unit-packets
			Path determination & Logical Addressing(IP)
			Devices/apps-router, firewall, layer 3 switch
			Protocols-ip,icmp, igmp
		Data Link
			Media Layers
			Data Unit-frames
			Physical Addressing(MAC && LLC)
			Devices/apps-bridge, layer 2 switch
			Protocols-rarp,arp(mac)
		Physical
			Media Layers	
			Data Unit-bits
			Media, Signal & Binary Transmission
			Devices/apps-hub
			Protocols-ethernet, token ring

Protocols
	A set of rules or formal specification to transmitting or receiving data
	It defines
		Format
		Timing
		sequence
		Error checking
		etc

	TCP			
		Reliable protocol
		Connection oriented
		Performas 3 ways handshake
		Provision of error detaction & retransmission
		FTP,HTTP, HTTPS
	UDP
		Unreliablle protocol
		COnnectionless
		Much faster than TCP
		No acknowledgement waits
		No proper seqnecing of data units
		SUitable where speed matters more than reliability
		DNS, DHCP,TFTP,ARP,RARP


	Protocols & default port numbers
		DNS		Domain Name Service-UDP		UDP 53
		DNS TCP	Domain Name Service-TCP		TCP 53
		Http		Web					TCP 80
		Https		Secure Web(SSL)			TCP 443
		SMTP		Simple Mail Transport		TCP 25
		POP		Post Office Protocol		TCP 109,110
		SNMP		Simple Network Management	TCP 161,162, UDP 161,162
		TELNET	Telnet terminal			TCP 23
		FTP		File transfer protocol		TCP 20,21
		SSH		Secure shell(terminal)		TCP 22
		AFP IP	Apple File Protocol/IP		TCP 447,548
